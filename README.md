This repository contains several tutorial notebooks to perform electron ptychography reconstructions with py4DSTEM.

It features several aspects of the data processing workflow, including: pre-processing, iterative reconstruction, parameter optimization and data visualization.

⚠️ Note: these notebooks are intended to run on a __GPU__.

Tested with py4DSTEM version __0.14.18__.

More information about py4DSTEM and how to use it for electron ptychography can be found here:
- py4DSTEM GitHub link -> https://github.com/py4dstem/py4DSTEM
- py4DSTEM tutorials link -> https://github.com/py4dstem/py4DSTEM_tutorials/tree/main/notebooks
